# PayTM QR Code Storage

Add your HelloQR.jpg file here.

The file will be used for PayTM payments in the BlockStop Upgrade page.

## Instructions:

1. Place your `HelloQR.jpg` in this folder
2. The system will automatically reference it as `Herey QR/HelloQR.jpg`
3. No need to commit - file is ignored by git (stays private)

## File Structure:
```
Herey QR/
├── .gitkeep (keeps folder in git)
├── .gitignore (ignores HelloQR.jpg)
├── README.md (this file)
└── HelloQR.jpg (your QR code - add here)
```
