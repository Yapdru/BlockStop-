IBlock brings BlockStop's powerful security analysis to your iPhone. Scan emails for phishing threats, analyze files for malware, and get real-time security insights in your pocket.

## Getting Started

### First Launch
1. Open IBlock and choose your authentication method:
   - Passkey (recommended - Face ID/Touch ID)
   - Google/Microsoft/Apple OAuth
   - Email + Password (less secure)

2. Select your tier:
   - FREE (default) - 50 free scans/month
   - Upgrade to NEO, PRO, or MAX for more features

3. Grant permissions:
   - Camera (for QR code scanning in Settings)
   - Files (to analyze documents)
   - Contacts (for email suggestions)

### Scanning Emails
1. Tap **Analyze** tab
2. Select **Email Check**
3. Paste email content or forward to analysis address
4. View instant threat analysis with:
   - Risk score (0-100)
   - Threat type (phishing, malware, spam)
   - Sender reputation
   - Suspicious links/attachments
   - Recommended actions

### Scanning Files
1. Tap **Analyze** tab
2. Select **File Scan**
3. Choose file from:
   - Files app
   - Cloud storage (Drive, OneDrive, Dropbox)
   - Email attachment
4. View analysis with:
   - File type and size
   - Threat level (Safe/Low/Medium/High/Critical)
   - Detected threats
   - Hash analysis
   - Remediation steps

### Using BetterBot AI (MAX only)
1. Tap **Chat** tab
2. Ask questions about your threats:
   - "What is this ransomware threat?"
   - "Should I click this link?"
   - "How do I remove this malware?"
   - "Explain this phishing email"
3. Get instant AI-powered security guidance

### Checking WiFi (NEO+ only)
1. Tap **WiFi Checker**
2. See nearby networks with security ratings
3. Tap to see:
   - Encryption type (WPA3/WPA2/WEP)
   - Signal strength
   - Security threats
   - Connection recommendations
4. Tap to connect to recommended networks

## Troubleshooting

### App Crashes on Launch
1. Force close the app (swipe up)
2. Clear app cache: Settings → IBlock → Offload App → Reinstall
3. Restart iPhone

### Scans Not Working
1. Check internet connection (WiFi or cellular)
2. Verify authentication is current
3. Try scanning again
4. Contact support if issue persists

### Biometric Auth Not Working
1. Check Face ID/Touch ID in Settings
2. Re-enroll biometric data if needed
3. Switch to password temporarily
4. Re-enable biometric when ready

### Billing Issues
1. Check Apple ID balance
2. Verify payment method in Settings
3. Try different payment method
4. Contact support with order ID

## FAQ

**Q: Is my data stored on BlockStop servers?**
A: Only your scan history is stored (with encryption). Email/file contents are never stored. You control retention.

**Q: Can I use multiple devices?**
A: Yes! Log in on any device with the same account. Your scan history syncs automatically.

**Q: What if I downgrade my tier?**
A: Premium features are disabled, but your data is preserved. Upgrade anytime to get them back.

**Q: How often are threat definitions updated?**

**Q: Can I export my scans?**
