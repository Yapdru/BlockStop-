## Getting Started

### Initial Setup
1. Launch IBlock and authenticate:
   - Passkey (Face ID/Touch ID recommended)
   - Google/Microsoft/Apple OAuth
   - Email + Password

2. Configure your workspace:
   - Set default team
   - Connect integrations (Slack, Teams, Jira)
   - Import threat feeds
   - Setup notifications

3. Enable features:
   - External keyboard shortcuts
   - Gestures and touch controls
   - Voice commands (available with MAX)

### Workspace Setup

**Split View for Maximum Productivity**
1. Open IBlock in Split View:
   - Email Analysis on left (80% width)
   - Browser research on right (20% width)
2. Use Apple Pencil to mark up details
3. Drag-drop evidence between apps

**Professional Dashboard**
1. Configure widgets:
   - Recent threats (5 most recent)
   - Team activity (real-time updates)
   - Threat distribution (pie chart)
   - Alert status (summary)

2. Customize shortcuts:
   - Quick email check (Cmd+E)
   - New file scan (Cmd+U)
   - Team chat (Cmd+T)
   - Advanced search (Cmd+F)

### Email Analysis Workflow

1. **Paste or Forward Email**
   - Copy email body → Analyze
   - Forward to analysis@blockstop.io
   - Drag from Mail app

2. **Review Threat Summary**
   - Risk score (0-100 scale)
   - Threat categories (phishing, malware, spam)
   - Sender reputation
   - Authentication check (SPF/DKIM/DMARC)

3. **Deep Dive**
   - Expand threat categories
   - Analyze suspicious links
   - Review attachment details
   - Check sender history

4. **Action & Report**
   - Mark as safe/suspicious
   - Add to blocklist
   - Share with team
   - Export PDF report

### File Scanning Workflow

1. **Import File**
   - Choose from Files app
   - Cloud storage (Drive, OneDrive, Dropbox)
   - Email attachment
   - USB/external drive

2. **Scan & Analyze**
   - YARA signature detection
   - Entropy analysis
   - API behavior analysis
   - ML threat scoring

3. **Review Results**
   - Threat detection list
   - File metadata
   - Hash analysis (MD5, SHA256)
   - MITRE ATT&CK mapping

4. **Remediation**
   - Quarantine file
   - Block hash globally (team)
   - Isolate infected system
   - Generate forensic report

## Troubleshooting

### Performance Issues
1. Close background apps (Cmd+Space, then close)
2. Reduce screen refresh rate if needed
3. Free up storage (delete old scans)
4. Restart iPad

### Sync Issues
1. Check internet connection
2. Sign out and sign back in
3. Re-open app
4. Contact support if persists

### Split View Problems
1. Ensure iPad OS 14+
2. Try closing and reopening
3. Disable Split View temporarily
4. Update app to latest version
