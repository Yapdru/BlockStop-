# IBlock - BlockStop for Apple TV

**Security Dashboard on Your TV**

## Overview

IBlock for tvOS brings BlockStop's threat analysis and team collaboration to your Apple TV. Perfect for security operations centers (SOC), incident response war rooms, and team dashboards showing real-time threat activity.

## Features

### SOC Dashboard
- 📊 **Real-Time Threat Feed** - Latest threats streaming on big screen
- 🎯 **Risk Metrics** - Team threat metrics and trends
- 👥 **Team Activity** - Who's investigating what
- 🔴 **Alert Status** - Critical incidents highlighted
- 📈 **Trend Analysis** - 7, 30, 90-day threat trends

### Incident War Room (NEO+)
- 🔍 **Investigation Timeline** - Visual event chronology
- 💬 **Team Chat** - Collaborate with audio/text
- 📋 **Task Board** - Track incident investigation tasks
- 📁 **Evidence Panel** - Threats and artifacts on screen
- 🎙️ **Voice Commands** - "Show critical incidents"

### Threat Visualization
- 🌍 **Geographic Heat Map** - Threats by location
- 📊 **Threat Distribution** - Pie charts by threat type
- 📈 **Timeline Charts** - Threat trends over time
- 🔗 **Correlation Map** - Related threats visualization
- 🎨 **Custom Themes** - Light/dark modes

### Remote Control
- 📱 **iPhone Control** - Use iPhone as remote
- ⌨️ **Keyboard Support** - USB/Bluetooth keyboard
- 🎮 **Apple Remote** - Native remote support
- 🖱️ **Trackpad** - Magic Trackpad compatible
- 🎤 **Siri** - Voice commands

## Getting Started

### Initial Setup

1. **Authenticate**
   - Use iPhone nearby for biometric auth
   - Or enter code from TV screen
   - Select your team

2. **Configure Display**
   - Choose dashboard layout (4, 6, or 9 sections)
   - Set refresh rate (5, 10, 30 seconds)
   - Enable sound alerts (optional)

3. **Add Team Members**
   - Invite analysts via email
   - Set notification levels
   - Configure who appears in feed

## Troubleshooting

### No Data Displaying
1. Check network connection
2. Verify WiFi signal
3. Sign out and sign back in
4. Restart Apple TV

### Refresh Too Slow
1. Increase refresh rate in settings
2. Check network speed
3. Reduce dashboard sections
4. Close other apps

### Sound Not Working
1. Check TV volume
2. Enable alerts in settings
3. Check notification volume
4. Restart app

### Remote Not Responding
1. Check Bluetooth battery
2. Re-pair remote
3. Restart Apple TV
4. Try using iPhone Remote app
