1. Create Your Account

1

Sign Up

Visit our web app and click "Get Started" to create your account. You can use:

  • Email address and password
  • Google account (OAuth)
2

Verify Your Email

Check your email inbox for a verification link. Click the link to confirm your account.

Tip: Check your spam folder if you don't see the email within a few minutes.
3

Choose Your Plan

Select the plan that best fits your needs:

  • FREE: Perfect for testing (10 scans/day)
  • NEO: Unlimited scans, desktop app ($9.99/month)
  • PRO: Everything + mobile app ($19.99/month)
  • MAX: Full enterprise features ($49.99/month)

2. Analyzing Emails with Email Checker

What is Email Checker? Our DRAR AI engine analyzes emails for phishing attempts, malicious links, spam indicators, and suspicious sender information.
1

Navigate to Email Checker

From the dashboard, click on "Email Checker" in the main menu.

2

Enter Email Address or Content

You can analyze emails in two ways:

  • Paste Email Content: Copy-paste the full email (headers included) into the text area
  • Enter Sender Email: Provide the sender's email address for reputation checking
3

Review Analysis Results

The report includes:

  • Risk Score: 0-100 threat level
  • Threat Categories: Phishing, spam, malware, etc.
  • Details: Specific indicators and recommendations
  • Sender Reputation: Historical sender analysis

Understanding Threat Scores

  • 0-20: Safe ✓
  • 21-50: Caution ⚠️ (low risk)
  • 51-80: Warning ⚠️ (medium risk)
  • 81-100: Danger ❌ (high risk)

3. Scanning Files with File Scanner

What is File Scanner? Our BetterBot PRO engine analyzes files for malware, trojans, ransomware, and other threats using signature and behavior analysis.
1

Navigate to File Scanner

From the dashboard, click on "File Scanner" in the main menu.

2

Upload a File

You can upload files in two ways:

  • Drag & Drop: Drag the file onto the upload area
  • Click to Browse: Click the upload area to select a file
Supported Formats: All common file types (ZIP, EXE, PDF, DOC, IMG, etc.)
3

Review Scan Results

The report includes:

  • File Information: Name, size, type, hash
  • Threat Level: Safe, Suspicious, or Dangerous
  • Detected Threats: Specific malware/virus names
  • Analysis Details: Entropy score, signatures matched

File Scanning Best Practices

  • Scan downloaded files before opening them
  • Scan email attachments immediately
  • Check archive contents before extracting
  • Review detailed reports for suspicious indicators

4. Setting Up Gmail Integration

Available on: NEO, PRO, and MAX tiers
1

Open Settings

Click the gear icon in the top-right corner of the dashboard.

2

Navigate to Integrations

In Settings, click on "Integrations" → "Gmail"

3

Connect Gmail Account

Click "Connect Gmail" and authorize BlockStop PRO to access your Gmail account. You'll be directed to Google's login page.

What permissions we request: Read emails, send labels, archive emails. We do NOT access passwords or send emails on your behalf.
4

Configure Scanning Settings

Choose how you want to handle threats:

  • Auto-scan: Automatically scan incoming emails
  • Auto-label: Label threats as "BlockStop Threat"
  • Auto-archive: Archive high-threat emails
  • Notifications: Get alerts for new threats
5

Save and Test

Click "Save Settings" and send a test email to verify integration is working.

5. Installing Desktop Application

Available on: NEO, PRO, and MAX tiers

macOS Installation

1

Download

Go to Settings → "Downloads" and click "Download macOS App"

2

Install

Open the downloaded .dmg file and drag BlockStop to Applications folder

3

Launch

Open Applications folder and click BlockStop to start the app

Windows Installation

1

Download

Go to Settings → "Downloads" and click "Download Windows App"

2

Run Installer

Run the .exe installer and follow the installation wizard

3

Login

Launch the app and sign in with your BlockStop account

Desktop App Features

  • Menu bar icon for quick access
  • Drag-drop file scanning
  • Keyboard shortcuts for quick scans
  • Desktop notifications for threats
  • Offline scanning capability

6. Installing Mobile Application

Available on: PRO and MAX tiers (iPhone)
1

Install from App Store

Open the Apple App Store and search for "BlockStop PRO"

2

Get the App

Click "Get" to install. You may need to authenticate with Face ID, Touch ID, or Apple ID

3

Launch and Login

Open BlockStop PRO and sign in with your account credentials

Mobile Features

  • Photo and file scanning from Photos app
  • Quick email threat analysis
  • Push notifications for threats
  • Offline scanning for cached results
  • Scan history access

7. Configuration & Settings

Account Settings

  • Profile: Update name, email, avatar
  • Password: Change your account password
  • Two-Factor Authentication: Enable 2FA for security
  • Session Management: View and manage active sessions

Scanning Settings

  • Auto-scan: Enable/disable automatic file scanning
  • Threat Sensitivity: Adjust detection strictness
  • File Type Filters: Choose which file types to scan
  • Max File Size: Set maximum file size for scanning

Notification Settings

  • Email Alerts: Get threat notifications via email
  • Push Notifications: Desktop/mobile alerts
  • Digest Mode: Summary emails of daily threats
  • Quiet Hours: Disable notifications during specific times

Privacy & Security

  • Data Retention: How long to keep scan history
  • Data Export: Export your data as JSON
  • Account Deletion: Permanently delete your account
  • Connected Apps: Manage app integrations

API Keys (PRO/MAX)

  • Generate API keys for programmatic access
  • View API documentation and integration guides
  • Monitor API usage and rate limits
